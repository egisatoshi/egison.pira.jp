Setting up Egison!!

How to install?

Egison is implemented in Haskell.
Egison is distributed as a Hackage program.

To install Egison, you need to install GHC and Cabal-install.
If you install Haskell Platform, both of GHC and Cabal-install will be installed.

After you installed Haskell Platform, perform following commands.

% cabal update
% cabal install egison
If successfully installed, there will be an output where the binary file is generated.
Please add new path to $PATH variable of your shell, to make your shell be able to find egison command.
(If your OS is Windows, you don't need to configure the path of the shell.)

After that, you can use Egison with the following command.

% egison

How to update?

You can update Egison with the following commands. 

    
% cabal update
% cabal install egison

Run Interpreter!!

After you have installed Egison, you can run Egison interpreter with a command 'egison'. (Setting up Egison!!)
Type Control-D to terminate the interpreter. 

  
% egison
Egison, version X.X.X : http://egison.pira.jp
Welcome to Egison Interpreter!
> ^D
Leaving Egison.
Byebye. See you again! (^^)/

How to use Egison mode on your emacs

To use Egison mode on your Emacs, you should copy a file egison-mode.el, which exists in ~/.cabal/share/, to load-path of your emacs, and add following lines in your .emacs. 

  
(autoload 'egison-mode "egison-mode" "Major mode for editing Egison code." t)
(setq auto-mode-alist
(cons `("\\.egi$" . egison-mode) auto-mode-alist))
