Example 1. Poker Hands

In this section, we will describe the program that determine the poker hands.
We do pattern matching collections of cards as a multiset of cards.
The program codes that we describe in this section can are in lib/poker-hands.egi and sample/poker-hands-test.egi.

First, we define the Card type.
To define it, we define the Suit type, and the Mod.

The definition of the Suit is as below.

  
(define $Suit
  (type
    {[$var-match (lambda [$tgt] {tgt})]
     [$inductive-match
      (destructor
        {[spade []
          {[<spade> {[]}]
           [_ {}]}]
         [heart []
          {[<heart> {[]}]
           [_ {}]}]
         [club []
          {[<club> {[]}]
           [_ {}]}]
         [diamond []
          {[<diamond> {[]}]
           [_ {}]}]
         })]
     [$=
      (lambda [$val $tgt]
        (match [val tgt] [Suit Suit]
          {[[<spade> <spade>] #t]
           [[<heart> <heart>] #t]
           [[<club> <club>] #t]
           [[<diamond> <diamond>] #t]
           [[_ _] #f]}))]
     }))
The definition of the Mod is as below.

  
(define $Mod
  (lambda [$m]
    (type
      {[$var-match (lambda [$tgt] {(mod tgt m)})]
       [$= (lambda [$val $tgt]
             (= (mod val m) (mod tgt m)))]})))

The Card type is defined using those two types defined above.

  
(define $Card
  (type
    {[$var-match (lambda [$tgt] {tgt})]
     [$inductive-match
      (destructor
        {[card [Suit (Mod 13)]
          {[<card $s $n> {[s n]}]}]})]
     [$= (lambda [$val $tgt]
           (match [val tgt] [Card Card]
             {[[<card $s $n>
                <card ,s ,n>]
               #t]
              [[_ _] #f]}))]}))

If we defined the Card type as abobe, we can define a function that determine poker hands as following.

  
(define $poker-hands
  (lambda [$Cs]
    (match Cs (Multiset Card)
      {[<cons <card $S $n>
         <cons <card ,S ,(- n 1)>
          <cons <card ,S ,(- n 2)>
           <cons <card ,S ,(- n 3)>
            <cons <card ,S ,(- n 4)>
             !<nil>
             >>>>>
        <straight-flush>]
       [<cons <card _ $n>
         <cons <card _ ,n>
          !<cons <card _ ,n>
            !<cons <card _ ,n>
              !<cons _
                !<nil>
                >>>>>
        <four-of-kind>]
       [<cons <card _ $m>
         <cons <card _ ,m>
          <cons <card _ ,m>
           <cons <card _ $n>
            !<cons <card _ ,n>
              !<nil>
              >>>>>
        <full-house>]
       [<cons <card $S _>
         !<cons <card ,S _>
           !<cons <card ,S _>
             !<cons <card ,S _>
               !<cons <card ,S _>
                 !<nil>
                 >>>>>
        <flush>]
       [<cons <card _ $n>
         <cons <card _ ,(- n 1)>
          <cons <card _ ,(- n 2)>
           <cons <card _ ,(- n 3)>
            <cons <card _ ,(- n 4)>
             !<nil>
             >>>>>
        <straight>]
       [<cons <card _ $n>
         <cons <card _ ,n>
          <cons <card _ ,n>
           <cons _
            <cons _
             !<nil>
             >>>>>
        <three-of-kind>]
       [<cons <card _ $m>
         <cons <card _ ,m>
          !<cons <card _ $n>
            <cons <card _ ,n>
             !<cons _
               !<nil>
               >>>>>
        <two-pair>]
       [<cons <card _ $n>
         <cons <card _ ,n>
          <cons _
           <cons _
            <cons _
             !<nil>
             >>>>>
        <one-pair>]
       [<cons _
         <cons _
          <cons _
           <cons _
            <cons _
             !<nil>
             >>>>>
        <nothing>]})))

We can use poker-hands function as follow.

  
> (test (poker-hands {<card <club> 4>
                      <card <club> 2>
                      <card <club> 13>
                      <card <club> 1>
                      <card <club> 3>}))
<straight-flush>
