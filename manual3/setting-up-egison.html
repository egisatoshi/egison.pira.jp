How to install?

Egison is implemented in Haskell and distributed as a Hackage program.

To install Egison, you need to install GHC and Cabal-install. If you have installed neither GHC nor Cabal-install, we recommend you to install Haskell Platform, which contains both.

After you installed Haskell Platform, perform following commands. 

% cabal update
% cabal install egison

...
Installing executable(s) in /home/xxx/.cabal/bin
Registering egison-3.0.0...

If successfully installed, there will be an output where the binary file is generated as above. Please add new path to $PATH variable of your shell, to make your shell be able to find egison command. (If your OS is Windows, you don't need to configure the path of the shell.) For example, if you use bash, you will execute commands as follows. 

% echo "PATH=\$PATH:/home/xxx/.cabal/bin" >> ~/.bashrc
% source ~/.bashrc

After that, you can use Egison with the following command. 

% egison

How to update?

You can update Egison with the following commands. 

% cabal update
% cabal install egison

Run Interpreter!!

After you have installed Egison, you can run Egison interpreter with a command 'egison'. Type Control-D to terminate the interpreter. 

% egison
Egison, version X.X.X : http://egison.pira.jp
Welcome to Egison Interpreter!
> ^D
Leaving Egison.
Byebye. See you again! (^^)/

How to use Egison mode on your emacs

To use Egison mode on your Emacs, you should copy a file egison-mode.el, which exists in ~/.cabal/share/, to load-path of your emacs, and add following lines in your .emacs. 

(autoload 'egison-mode "egison-mode" "Major mode for editing Egison code." t)
(setq auto-mode-alist
(cons `("\\.egi$" . egison-mode) auto-mode-alist))
