Example : Poker Hands

In this section, we will describe the program that determine the poker hands.
We do pattern matching collections of cards as a multiset of cards.
The program codes that we describe in this section can are in sample/poker-hands-test.egi.

First, we define the card matcher.
To define it, we define the suit matcher, and the mod.

The definition of the suit is as below.

(define $suit
  (algebraic-data-matcher
    {<spade> <heart> <club> <diamond>}))

The definition of the mod is as below.
This is included in "lib/core/number.egi".

(define $mod
  (lambda [$m]
    (matcher
      {[,$n []
        {[$tgt (if (eq? (modulo tgt m) (modulo n m))
                   {[]}
                   {})]}]
       [$ [something]
        {[$tgt {(modulo tgt m)}]}]
       })))

The card matcher is defined using those two types defined above.

(define $card
  (algebraic-data-matcher
    {<card suit (mod 13)>}))

If we defined the card matcher as above, we can define a function that determine poker hands as following.

(define $poker-hands
  (lambda [$cs]
    (match cs (multiset card)
      {[<cons <card $s $n>
         <cons <card ,s ,(- n 1)>
          <cons <card ,s ,(- n 2)>
           <cons <card ,s ,(- n 3)>
            <cons <card ,s ,(- n 4)>
             <nil>>>>>>
        <Straight-Flush>]
       [<cons <card _ $n>
         <cons <card _ ,n>
          <cons <card _ ,n>
            <cons <card _ ,n>
              <cons _
                <nil>>>>>>
        <Four-of-Kind>]
       [<cons <card _ $m>
         <cons <card _ ,m>
          <cons <card _ ,m>
           <cons <card _ $n>
            <cons <card _ ,n>
              <nil>>>>>>
        <Full-House>]
       [<cons <card $s _>
         <cons <card ,s _>
           <cons <card ,s _>
             <cons <card ,s _>
               <cons <card ,s _>
                 <nil>>>>>>
        <Flush>]
       [<cons <card _ $n>
         <cons <card _ ,(- n 1)>
          <cons <card _ ,(- n 2)>
           <cons <card _ ,(- n 3)>
            <cons <card _ ,(- n 4)>
             <nil>>>>>>
        <Straight>]
       [<cons <card _ $n>
         <cons <card _ ,n>
          <cons <card _ ,n>
           <cons _
            <cons _
             <nil>>>>>>
        <Three-of-Kind>]
       [<cons <card _ $m>
         <cons <card _ ,m>
          <cons <card _ $n>
            <cons <card _ ,n>
             <cons _
               <nil>>>>>>
        <Two-Pair>]
       [<cons <card _ $n>
         <cons <card _ ,n>
          <cons _
           <cons _
            <cons _
             <nil>>>>>>
        <One-Pair>]
       [<cons _
         <cons _
          <cons _
           <cons _
            <cons _
             <nil>>>>>>
        <Nothing>]})))

We can use poker-hands function as follow.

> (poker-hands {<Card <Heart> 1>
                <Card <Heart> 2>
                <Card <Heart> 3>
                <Card <Heart> 4>
                <Card <Heart> 5>})
<Straight-Flush>
> (poker-hands {<Card <Club> 4>
                <Card <Club> 2>
                <Card <Club> 13>
                <Card <Club> 1>
                <Card <Club> 3>})
<Flush>
> (poker-hands {<Card <Diamond> 1>
                <Card <Club> 2>
                <Card <Club> 1>
                <Card <Heart> 1>
                <Card <Diamond> 2>})
<Full-House>
> (poker-hands {<Card <Diamond> 4>
                <Card <Club> 2>
                <Card <Club> 5>
                <Card <Heart> 1>
                <Card <Diamond> 3>})
<Straight>
> (poker-hands {<Card <Diamond> 4>
                <Card <Club> 10>
                <Card <Club> 5>
                <Card <Heart> 1>
                <Card <Diamond> 3>})
<Nothing>
