What is Egison?

Egison is the

  • pattern-matching-oriented
  • purely functional
programming language.

A feature of Egison is the strong pattern matching facility. With Egison, you can represent pattern matching for data types whose data have no normal forms, such as set, multiset, and algebraic structures such as group, ring, field, and so on.

Pattern Matching of Egison

Egison can pattern-match with data types whose data have no canonical forms. Such kind of data types is called an unfree data type.

In pattern-matching of well-known programming languages, the existence of canonical forms is assumed. Therefore, when we do pattern-matching with unfree data types, we need to translate and regard them as data which have a canonical form. For example, a set would be treated as a list. Many programmers think this is unavoidable, and in fact, it is a latent stress of programming. Sets and multisets often play important roles in expressing algorithms. So natural expression for handling these kinds of data is also important.

Egison is the world's first programming language that makes pattern-matching with unfree data types practical. Following features enables practical pattern-matching with unfree data types.

Multiple pattern-matching results
Pattern-matching has multiple results. This feature is necessary to pattern-match with unfree data types. Because unfree data have many forms for same data, so there are multiple ways of destruction.
Non-linear patterns
Bindings in a pattern can be refer in the same pattern. It is necessary to write useful patterns for pattern-matching with multiple results.
Patterns have lexical scoping
We can modularize usefule patterns with pattern-functions, functions that receive patterns and return a pattern. Since a pattern-function has lexical scoping, bindings for pattern-variables in the argument patterns and the body of pattern-functions don't conflict. Useful patterns can be reused in many places in a program without worry of name clash.

Egison's pattern-matching is going to include a following feature.

Auto-parallelized pattern-matching
Parallelizable loops are expressed with a single pattern-matching expression in Egison. Pattern-matching process inside Egison itself can be parallelized easily.

Algorithms are expressed intuitively and will be auto-parallelized with Egison!!
