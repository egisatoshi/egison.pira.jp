What is Egison?

Egison is the pattern-matching-oriented pure functional programming language.
A feature of Egison is the strong pattern match facility.
With Egison, you can directly represent pattern matching for data types which data have no normal forms, such as set, multiset, and algebraic structures such as group, ring, field, and so on.
